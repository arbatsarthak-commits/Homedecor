# 🏠 Jiyu's HomeDecor

A modern and responsive Home Decor website built using HTML, CSS, and JavaScript. The project provides an elegant user interface for browsing premium home decor products, managing a shopping cart, and completing the checkout process.

---

## 📌 Project Overview

Jiyu's HomeDecor is an e-commerce website designed for home decor enthusiasts. It offers a seamless shopping experience with an attractive UI, product listings, cart management, and checkout functionality.

The website focuses on:

- Premium home decor products
- User-friendly design
- Responsive layouts
- Smooth navigation between pages
- Modern UI with animated backgrounds and gradients

---

- Beautiful and modern landing page
- Product catalog with attractive UI

## 💻 How to Run the Project

### Method 1: Using VS Code

1. Download or clone the repository.

```bash
git clone <repository-link>
```

2. Open the project in Visual Studio Code.

3. Install the Live Server extension.

4. Right-click on `index.html`.

5. Click:

```
Open with Live Server
```

---

### Method 2: Run Directly

Simply open:

```
index.html
```

in any modern browser such as:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
